# Soft Dis-Play — ゲームシステム設計書

## コンセプト

> 「その悪口、もう少しお上品に言えませんか？」

**悪口をいかに知的にマイルドへ変換するか**を競う、言語センス×カード戦略のオンラインパーティーゲーム。  
直接的な悪口に「クッション言葉」を組み合わせ、最もセンスある表現をした人が勝ちを掴む。

参考雰囲気：ボディビル大会ボードゲームのような**まじめにやるのがおかしい**競技感。

---

## 基本情報

| 項目 | 内容 |
|---|---|
| プレイ人数 | 2〜4人 |
| プレイ形式 | オンラインマルチプレイ（各自デバイスを持つ） |
| 1ゲームの目安 | 15〜45分 |
| ゲーム終了条件 | ホストが作成時に指定したラウンド数（1〜10）の消化 |

---

## カードの種類

### 悪口カード（罵倒デッキ）

文章の「オチ」を担当する、ビジネスシーンにおけるネガティブな状態や評価を表すカード。全40枚以上（初期構成40枚）収録。
「〜が」「〜は」などの主語を含まない「状態」「比喩」「評価」に特化。

| カテゴリ | 採用例 |
|---|---|
| 状態 | 「完全に観光気分」「野放しの牧場主状態」「ただの迷子のお知らせ」 |
| 成果物評価 | 「もはや怪文書」「永遠のベータ版」「謎のブラックボックス」 |
| 比喩 | 「大渋滞」「スピード違反」「燃え盛る火の車」 |
| 評価 | 「控えめに言ってポンコツ」「圧倒的な丸投げ」「無駄の極み」 |

> 🚫 **除外対象**：外見・体型・容貌・社会属性・性別・出自に関するすべての表現

---

### クッション言葉カード（クッションデッキ）

文章の冒頭に置き、直接的な悪口を「丁寧な響き」に中和するカード。全30枚以上（初期構成30枚）収録。

| カテゴリ | 採用例 |
|---|---|
| 依頼・配慮系 | 「恐れ入りますが」「お手数ではございますが」 |
| 指摘・反論系 | 「お言葉を返すようですが」「僭越ながら」 |
| 肯定風 | 「ご意見として大変興味深いのですが」「チャレンジ精神は素晴らしいのですが」 |
| 遠回し | 「率直に申し上げにくいのですが」「語弊を承知で申し上げますと」 |

---

### 名詞カード（名詞デッキ）

ビジネス、場面、概念などの文脈を作る単語。全50枚以上（初期構成50枚）収録。  
例：「社長」「このエクセル」「定例ミーティング」「社長の思いつき」「マイクのミュート」など。

---

### 助詞カード（助詞デッキ）

カード同士を繋げて自然な文章にするための接着剤。全50枚以上（初期構成50枚）収録。
通常の助詞に加え、文章を収束させる「文末表現」や、展開を作る「接続副詞」を追加。  
例：「は」「が」「を」 / 「逆に」「控えめに言って」 / 「…ですね」「…の極みです」など。

## ゲームの流れ

### ゲーム開始時

1. ルーム作成（ホストがラウンド数を1〜10で設定） / 参加（オンライン）
2. 全4種類のデッキをシャッフル
3. 各プレイヤーに **初期手札計24枚（悪口4、クッション4、名詞8、助詞8）** を配る

---

### 各ラウンドの流れ

```
[ フェーズ1：交換フェーズ ]
　↓
[ フェーズ2：選択フェーズ ]
　↓
[ フェーズ3：発表フェーズ ]
　↓
[ フェーズ4：投票フェーズ ]
　↓
[ フェーズ5：採点フェーズ ]
```

#### フェーズ1：交換フェーズ（手札マネジメント）

- 各プレイヤーは手札から**任意のカードを最大2枚まで**選び、対応するデッキから新しいカードとランダムに交換できる
- 交換は任意（0枚で「準備完了」も可）
- 全員が「準備完了」になると次へ進む

#### フェーズ2：選択フェーズ（秘密）

- 手札から任意のカードを好きな枚数選び、**ドラッグ＆ドロップで文章ビルダーに並べる**
- 悪口、クッション、名詞、助詞を自然に組み合わせて1つの文章を完成させる
- 他のプレイヤーには非表示
- 全員が提出すると次へ進む

#### フェーズ3：発表フェーズ

- 全員が選択を終えたら **同時に公開**
- 表示例：

```
（例：クッション + 名詞 + 助詞 + 悪口）
「恐れ入りますが」「部長」「は」「のろま」
```

- 発表の順番はランダム（誰が作ったかは隠したまま投票へ）

#### フェーズ4：投票フェーズ
- 各プレイヤーは **自分以外** の表現から1つに投票
- 投票基準は自由（ルール画面に以下を表示）：

> 💡 **投票のヒント**
> - 一番マイルドに聞こえる表現に入れてもOK
> - 一番笑えた・センスを感じた表現に入れてもOK
> - あなたの感覚で「これが一番うまい！」と思ったものへ

- タイムリミット：**30秒**

#### フェーズ5：採点フェーズ

- **最多得票者** に1点
- 同票の場合は該当者全員に1点
- 使用したカードは消費される
- 次のラウンドへ（設定ラウンド数に到達した場合はゲームオーバー画面へ移行）

---

## 勝利条件

- 全ラウンド終了時に**最もポイントが多いプレイヤーが優勝**
- 同点の場合は「サドンデスラウンド」（1ラウンド追加）

---

## UIの主要画面

| 画面 | 内容 |
|---|---|
| ロビー | 部屋作成・参加、ラウンド数設定 |
| 手札画面 | 自分の手札表示、交換操作 |
| 選択画面 | 出すカードを選ぶ（他者には非表示） |
| 発表画面 | 全員の組み合わせを一覧表示 |
| 投票画面 | 好きな組み合わせに投票 |
| 結果画面 | 得票数、ポイント、ラウンド結果表示 |
| スコアボード | 累計ポイント、ゲーム終了時の最終結果 |

## 技術要件（参考）
