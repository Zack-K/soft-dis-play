# ソフト・ディス・プレイ 開発・改善ログ

## 2026-02-24: 語彙構成の刷新とカード枚数の拡張

ゲームデザイナーとしての検討に基づき、文章作成の難易度低下とゲームの安定性向上を目的とした大規模なアップデートを実施しました。

### 1. 変更の背景と目的
- **文章構築の破綻解消**: 悪口カードが主語（「仕事が〜」等）を含んでいたため、名詞カードと組み合わせた際に文法が崩れる問題を「フレーズの汎用化」で解決。
- **着地（オチ）の用意**: 文章を締める語尾（「…ですね」等）を助詞カード枠に追加し、ビジネス構文としての完成度を向上。
- **数学的エラーの回避**: 最大4人プレイ時の初期配布枚数がデッキ総数を上回るリスクを、170枚への増量により解消。

### 2. アップデート内容

| カード種別 | 枚数 | 主な変更点 |
| :--- | :--- | :--- |
| **Cushion (クッション)** | 30枚 | ビジネスシーンで使いやすい丁寧な前置きのバリエーション拡充。 |
| **Insult (悪口/オチ)** | 40枚 | 主語を排し、「もはや怪文書」「永遠のベータ版」等の汎用フレーズへ刷新。 |
| **Noun (名詞)** | 50枚 | 「このエクセル」「マイクのミュート」等、解像度の高いビジネス語彙を追加。 |
| **Particle (助詞/接続詞)** | 50枚 | 文末表現（…の極みです 等）や、接続詞（逆に、もはや 等）を追加。 |

### 3. 検証結果
- `cards.json` のJSONパースおよび枚数チェック（全170枚）を確認済み。
- `docs/game_design.md` への仕様反映を完了。
