# Soft Dis-Play — ゲームシステム設計書

## コンセプト

> 「その悪口、もう少しお上品に言えませんか？」

**悪口をいかに知的にマイルドへ変換するか**を競う、言語センス×カード戦略のオンラインパーティーゲーム。  
直接的な悪口に「クッション言葉」を組み合わせ、最もセンスある表現をした人が勝ちを掴む。

参考雰囲気：ボディビル大会ボードゲームのような**まじめにやるのがおかしい**競技感。

---

## 基本情報

| 項目 | 内容 |
|---|---|
| プレイ人数 | 2〜4人 |
| プレイ形式 | オンラインマルチプレイ（各自デバイスを持つ） |
| 1ゲームの目安 | 30〜45分 |
| ゲーム終了条件 | 固定ラウンド数消化（推奨：プレイヤー人数 × 2ラウンド） |

---

## カードの種類

### 悪口カード（罵倒デッキ）

`日本語悪口語彙辞典.md` をベースにカード化。

以下カテゴリを使用。**差別的・侮蔑的・美醜に関する表現はすべて除外する。**

| カテゴリ | 採用例 |
|---|---|
| 知能・能力系 | ぼんくら、のろま、ポンコツ、石頭 |
| 性格・行動系 | ケチ、怠け者、嘘つき、偽善者、見栄っ張り |
| 比喩系（行動・能力） | 雑魚、木偶の坊、能なし |
| 古風系 | おたんこなす、とんちんかん、すっとこどっこい |

> 🚫 **除外対象**：外見・体型・容貌・社会属性・性別・出自に関するすべての表現

---

### クッション言葉カード（クッションデッキ）

`クッション言葉一覧.md` + `婉曲表現一覧.md` をベースにカード化。  
**英語の婉曲表現は自然な日本語訳に変換してカード化する（英語原文は使用しない）。**

| カテゴリ | 採用例 |
|---|---|
| 依頼・配慮系（ビジネス優先） | 「恐れ入りますが」「お手数ですが」「大変僭越ながら」 |
| 断り・謝罪系 | 「誠に申し訳ございませんが」「せっかくではございますが」 |
| 反論・指摘系 | 「お言葉を返すようですが」「念のため申し上げますと」 |
| 婉曲系（京言葉） | 「さすがですなぁ」「えらいご熱心どすな」「まぁまぁ、よう似合ってますわ」 |
| 婉曲系（英語→日本語訳） | 「それはまた大胆なご決断ですね」「なかなか印象的でございますね」 |

> 英語原文（"That's a bold decision" 等）をそのまま使うのではなく、**同じ婉曲ニュアンスを持つ自然な日本語訳**をカードに記載する。

## ゲームの流れ

### ゲーム開始時

1. ルーム作成 / 参加（オンライン）
2. 悪口デッキ・クッション言葉デッキをシャッフル
3. 各プレイヤーに **悪口カード3枚 + クッション言葉カード3枚** を配る
4. **ラウンド数を決定**（例：人数 × 2）

---

### 各ラウンドの流れ

```
[ フェーズ1：交換フェーズ ]
　↓
[ フェーズ2：選択フェーズ ]
　↓
[ フェーズ3：発表フェーズ ]
　↓
[ フェーズ4：投票フェーズ ]
　↓
[ フェーズ5：採点フェーズ ]
```

#### フェーズ1：交換フェーズ（手札マネジメント）

- 各プレイヤーは **1ラウンドに1回**、好きなカードを最大2枚まで山札と交換できる
- 交換は任意（スキップ可）
- 交換制限を使い切った場合はそのラウンドは交換不可
- タイムリミット：**30秒**

#### フェーズ2：選択フェーズ（秘密）

- 手札から **悪口カード1枚 + クッション言葉カード1枚** を選ぶ
- 他のプレイヤーには非表示
- タイムリミット：**60秒**

#### フェーズ3：発表フェーズ

- 全員が選択を終えたら **同時に公開**
- 表示例：

```
┌──────────────────────────────┐
│  大変僭越ながら、           │
│  あなたは のろま でいらっしゃる │
└──────────────────────────────┘
```

- 発表の順番はランダム（誰が作ったかは隠したまま投票へ）

#### フェーズ4：投票フェーズ

- 各プレイヤーは **自分以外** の表現から1つに投票
- 投票基準は自由（ルール画面に以下を表示）：

> 💡 **投票のヒント**
> - 一番マイルドに聞こえる表現に入れてもOK
> - 一番笑えた・センスを感じた表現に入れてもOK
> - あなたの感覚で「これが一番うまい！」と思ったものへ

- タイムリミット：**30秒**

#### フェーズ5：採点フェーズ

- **最多得票者** に1点
- 同票の場合は該当者全員に1点
- 使用したカード2枚は捨て札へ
- 手札を補充（デッキが切れたらシャッフルして再利用）
- 次のラウンドへ

---

## 勝利条件

- 全ラウンド終了時に**最もポイントが多いプレイヤーが優勝**
- 同点の場合は「サドンデスラウンド」（1ラウンド追加）

## UIの主要画面

| 画面 | 内容 |
|---|---|
| ロビー | 部屋作成・参加、ラウンド数設定 |
| 手札画面 | 自分の手札表示、交換操作 |
| 選択画面 | 出すカードを選ぶ（他者には非表示） |
| 発表画面 | 全員の組み合わせを一覧表示 |
| 投票画面 | 好きな組み合わせに投票 |
| 結果画面 | 得票数、ポイント、ラウンド結果表示 |
| スコアボード | 累計ポイント、ゲーム終了時の最終結果 |
